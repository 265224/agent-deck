# Agent Deck Project Docs

This folder is the clean documentation workspace for the Agent Deck app folder.
It contains the active project summary, build notes, integration notes, and
desktop pet mode notes. Historical source snapshots are intentionally excluded
so this folder stays aligned with the Agent Deck name.

## Contents

- `summaries/01-project-overview.md`: current folder layout and app identity
- `summaries/02-build-run-verify.md`: local build, run, and verification notes
- `summaries/03-architecture-flow.md`: package targets and runtime flow
- `summaries/04-integrations-hooks.md`: supported agent hook integration notes
- `summaries/05-desktop-pet-mode.md`: desktop pet mode behavior and assets
- `summaries/06-assets.md`: bundled assets and source asset locations

## Canonical App Identity

- App name: `Agent Deck`
- Local bundle: `~/Applications/Agent Deck.app`
- Bundle id: `app.agentdeck.local`
- Swift package: `AgentDeck`
- App executable: `AgentDeckApp`
- Hook helper: `AgentDeckHooks`
- Setup helper: `AgentDeckSetup`
